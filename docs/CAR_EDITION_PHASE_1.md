# 车机版第一阶段说明

本阶段提供一个可单独安装的 Android 车机 APK，面向已经确认可以安装第三方 APK 的车机。

## 已实现

- 独立应用包名：`cn.toside.music.mobile.car`，可与手机版同时安装。
- 启动后固定横屏，并请求 Android 沉浸式全屏；同时声明不可调整窗口，避免被普通 Android 多窗口模式缩小。
- 复用应用已有横屏界面、播放能力及私有同步服务；包括本 Fork 的自定义音源同步。
- 构建保留 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 和通用 APK。首次安装优先选择通用 APK。

## 获取与安装

1. 打开仓库的 **Actions**，进入成功的 `Build Test APK` 工作流。
2. 下载 `lx-music-car-test-<commit>` 工件并解压。
3. 在车机上安装文件名含 `universal.apk` 的 APK。
4. 首次启动应直接进入横屏界面；在「设置 → 数据同步服务」中填入你的同步服务地址和账号。

测试 APK 使用调试签名但内置 JavaScript bundle，因此不依赖电脑或 Metro 开发服务器。若车机曾安装相同包名但签名不同的旧车机 APK，需要先卸载旧车机 APK 再安装。

## 适用范围与后续

WEY VV5 2020 等原车机是否允许安装第三方 APK 取决于系统版本和车辆配置。本构建不尝试绕过车机系统限制；仅在你已能正常安装 APK 的前提下使用。

如果车机桌面仍将应用固定在一个小窗口内，则属于桌面/系统对第三方应用的窗口策略，应用无法绕过。此时本版仍会隐藏应用自身的状态栏和导航栏，但不能占用车机桌面的其它卡片区域。

第二阶段再针对实际使用体验补充遥控器/旋钮焦点、较大触控目标和车机屏幕分辨率适配。
